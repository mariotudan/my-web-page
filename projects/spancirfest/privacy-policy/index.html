Privacy Policy

English
1. Introduction
This Privacy Policy governs the use of Špancirfest mobile application for devices with Android Lollipop or higher, which is created by Mario Tuđan.

This application provides information about Špancirfest. All information shown on a mobile application is also available on the Špancirfest website. Available information in the application is listed below:

- Program
- Interactive map with points of interest
- Image gallery from the official Špancirfest public album
- Quiz in cooperation with Tourist Board of Varaždin

2. Permissions while using the application
While using Špancirfest application, the application will need permissions for some feature. The user can choose to grant or deny permissions, but if permission is denied, some application functionalities may not be available. Permission needed for some application functionalities are:

- Permission to access the camera, to give a user a possibility to scan QR quiz codes
- Permission to write on external storage, to give user possibility to save images from the gallery and quiz results
- Permission to access user's location, to give user possibility to see user on map

3. Data collection
Špancirfest application can collect usage data while an application is in use and send crash reports, if the user agrees to enable those features. When an application is opened the first time, the user is asked if he wants to give permission to the application to collect usage data and send crash reports. The user can change that anytime in application settings.

3.1. Data usage collection
This application uses Firebase Analytics as a service for collecting usage data. The špancirfest application collects usage data and a consumption of content which is provided to the user, which also implies to get information about all content visited by the user inside the application.

At the first launch of an application, the user has a choice to give permission to the application to collect usage data. In case of giving the permission, the application will collect usage data, while if permission is not granted, data collection will be disabled. The user can change an option of data collection in application settings anytime.

Firebase Analytics automatically collects additional information while an application is in use, and those data are:

- Internet address
- Geolocation
- Device operating system
- Device model
- Number of users and sessions
- Session duration
- Unique device identifier
- First launches
- App opens
- App updates

Data is retained for 14 months, and after that is invalidated. If the user wants to delete its data earlier, this can be done by reset option in application settings.

3.2. Crash reporting
This application can send crash reports and system used for this functionality is Firebase Crashlytics. Automatic crash reporting gives developer detailed information about application issues and with that kind of reports, it's easier to detect and resolve common application issues which users are facing.

At the first launch of the application, the user has a choice to give permission to the application to send crash reports. In case of giving the permission, the application will send crash reports if one occurs, while if permission is not granted, crash reporting will be disabled. The user can change an option of crash reporting in application settings anytime.

Firebase Crashlytics collects data when sending crash reports, and those are:

- Installation UUID
- Crash traces

Crash traces and their associated identifiers are kept for 90 days.

4. Data usage
Špancirfest collects data of application usage to give a possibility to make the application better. Collected data gives insights into general usage od the application which leads to better usage understanding and to give users even better application experience.

Collected data will be used by Tourist Board of Varaždin for a better understanding of how the application is used in general and which content is most popular in general. Collected data will help Špancirfest in the future because it will give insights in content usage based on general users usage.

5. User choices and opt-out
At application launch, the user can choose to grant or deny to the application to collect usage data and to send crash reports. That choice is always available in application settings, and the user can change it anytime.

If the user wants to invalidate its collected data, in application settings is available reset option, which will invalidate all collected data from unique identifier used for data collection.

6. Agreement
By using Špancirfest mobile application, you agree to this Privacy Policy.

7. Contact
If you have any question, you can contact me via email: mario@tuđan.com



Politika privatnosti

Hrvatski
1. Uvod
Ova pravila privatnosti reguliraju upotrebu mobilne aplikacije “Špancirfest” za uređaje s operacijskim sustavom Android Lollipop ili noviji, koju je izradio Mario Tuđan.

Ova aplikacija pruža informacije o Špancirfest festivalu, a sve informacije prikazane u aplikaciji nalaze se i na web stranici festivala. Informacije koje se nalaze u aplikaciji su:

- Program festivala
- Interaktivna karta lokacija
- Galerija slika sa službenog Špancirfest album
- Kviz u suradnji s Turističkom zajednicom grada Varaždina

2. Dopuštenja pri korištenju aplikacije
Tijekom korištenja Špancirfest aplikacije, aplikacija traži dopuštenja koja je moguće odobriti ili odbiti, s time da u slučaju odbijanja davanja dopuštenja neke funkcionalnosti aplikacije neće biti dostupne. Tražena dopuštenja tijekom korištenja i razlog zbog kojeg su potrebna su:

- Dopuštenje za pristup kameri uređaja radi skeniranja QR kodova kviza
- Dopuštenje za pristup pohrani podataka zbog mogućnosti spremanja slika iz galerije i rezultata kviza
- Dopuštenje za pristup lokacije korisnika zbog mogoćnosti prikazivanje korisnikove lokacije na karti

3. Prikupljanje podataka
Špancirfest aplikacija tijekom korištenja može prikupljati podatke o korištenju te slati izvješća o ispadu aplikacije ukoliko korisnik to odobri. Korisnik je na početku aplikacije upitan da li želi dati odobrenje za korištenje prikupljanja podataka i mogućnosti prikupljanja izvještaja o ispadu. U bilo kojem trenutku, korisnik u postavkama aplikacije može promijeniti svoj odabir.

3.1. Analitika korištenja aplikacije
Ova aplikacija za prikupljanje podataka koristi servis Firebase Analytics. Špancirfest aplikacija prikuplja podatke o korištenju aplikacije te konzumaciji sadržaja koji je pružen korisniku, odnosno na koji način koristite aplikaciju što podrazumijeva posjećivanje raznih sadržaja unutar aplikacije.

Pri prvom pokretanju aplikacije, korisniku je dan izbor da li želi, odnosno ne želi dati aplikaciji pristup da prikuplja podatke o korištenju aplikacije. U slučaju davanja pristupa aplikacija će prikupljati podatke o korištenju aplikacije, dok će u protivnom prikupljanje podataka biti onemogućeno. U svakom trenutku korisnik može promijeniti izbor odnosno promijeniti mogućnost prikupljanja podataka u postavkama aplikacije.

Sustav Firebase Analytics automatizirano prikuplja dodatne informacije pri korištenju aplikacije, a ti podaci su:

- Internet adresa
- Geolokacija
- Operacijski sustav uređaja
- Model uređaja
- Broj korisnika i sesija
- Trajanje sesije
- Jedinstveni identifikator uređaja
- Prvo korištenje aplikacije
- Otvaranje aplikacije
- Ažuriranje aplikacije

Navedeni podaci čuvaju se u sustavu Firebase Analytics do 14 mjeseci, nakon čega se automatski invalidiraju. Ukoliko korisnik želi ranije obrisati svoje podatke, moguće ih je resetirati u postavkama aplikacije.

3.2. Automatsko slanje izvještaja o ispadu aplikacije
Ova aplikacija omogućuje automatsko slanje izvještaja o ispadu aplikacije, a sustav korišten za to je Firebase Crashlytics. Automatsko slanje izvještaja o ispadu aplikacije omogućuje uvid u trenutne probleme aplikacije s kojima se korisnici susreću, te se na temelju izvještaja navedeni problemi mogu riješiti.

Pri prvom pokretanju aplikacije korisnik može odabrati da li želi odnosno ne želi dati aplikaciji mogućnost da automatski šalje izvještaje o ispadu aplikacije. U svakom trenutku moguće je promijeniti izbor odnosno promijeniti mogućnost prikupljanja podataka u postavkama aplikacije.

Sustav Firebase Crashlytics prikuplja određene podatke pri slanju informacija o ispadu aplikacije, a to su:

- Jedinstveni identifikator (UUID)
- Podaci o ispadu

Podaci o ispadu i jedinstveni identifikatori zadržavaju se u sustavu devedeset (90) dana.

4. Korištenje podataka
Špancirfest prikuplja podatke o korištenju aplikacije radi poboljšanja aplikacije, odnosno kako bi se na temelju korištenja aplikacije moglo vidjeti koji sadržaji aplikacije bi mogli biti poboljšani na način da korisniku sadržaj bude dostupniji.

Podaci o korištenju aplikacije biti će korišteni od strane Turističke zajednice grada Varaždina kako bi na temelju generalnog korištenja aplikacije mogli vidjeti kako su korisnici koristili aplikaciju u smislu konzumiranja sadržaja Špancirfest festivala radi daljnjih poboljšanja u programu, pristupu i pripremi nadolazećih festivala.

5. Vaše mogućnosti i izbor
Korisnik pri pokretanju aplikacije može odabrati želi li odnosno ne želi dati dopuštenje za prikupljanje podataka te automatizirano slanje izvještaja o ispadu aplikacije. Navedeni izbor uvijek je moguće promijeniti u postavkama aplikacije.

U slučaju da korisnik želi invalidirati svoje prikupljene podatke, u postavkama aplikacije moguće je odabrati resetiranje prikupljenih podataka, pri čemu će se svi podaci povezani s jedinstvenim identifikatorom invalidirati sa sustava za prikupljanje podataka.

6. Sporazum
Korištenjem ove aplikacije prihvaćate uvjete politike privatnosti.

7. Kontakt
